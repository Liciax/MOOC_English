1 - What is computer science ?

Computer science is the automatic information processing by machines.

A definition: "Information is information, no matter nor energy" Wiener 1968.

2 - What is algorithmic ?

An algorithm is writing in an unambiguous language of a method for solving a problem. This method is likely to be encoded on a computer.

It is necessary to write good algorithms to reduce runtime (time complexity) and memory size (space complexity).

The algorithm (paper version) is written by an analyst and executed by a contractor. The user gives the values and receives the results.

The program (machine version) is coded by a programmer and executed by a computer. The user types on the keyboard and the screen reads.

Examples of algorithm in real life : recipe, installation instructions...

An algorithm has 4 parts:

  • Header
  • Statement
  • Actions
  • Comments

Here is an example of algorithm syntax :

Algorithm A_title //header

variables //statement
  name_var1 : type

Beginning //actions
  instructions
  //one comment
  ...
End

The two bases of the algorithms :

  • The variables, in which it stores the information (temporary storage area).
  • The instructions for changing the status of these variables.
