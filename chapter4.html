Part 4 - Boolean logic

1 - Logical operators

The boolean type variables may contain as value "true" of "false". This value can be the result of a expression. A expression is a value, or a variable name, or is an operator applied to values or variable names.

The comparison operators are : <, >, =, <=, >=, !=

2 - Truth tables

This is the results of the most common boolean expression that you need to know (no, and, or).

True False
False True
A B A and B

A B A or B
