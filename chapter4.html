Part 4 - Boolean logic

When we have two values, it is possible to compare one with the other. Is the first one bigger than the second one? Is it longer? Are they of the same type? Such a comparaison has an answer : yes or no. In algorithm this answer is a boolean.

1 - Logical operators

The boolean type variables may contain as value "true" of "false". Is is the answer of a comparaison between two or more values with a comparaison operator.

The comparison operators are : <, >, ==, <=, >=, !=

with this, we can compare two values and get an answer: "X is bigger than Y" would produce the value "true" if X>Y, or the value "false" otherwise. Let's try it with some code :

console.log(6>3);

we can see that this is true: 6 in indeed superior to 3. on the other hand :

console.log(6==3);

is false. Since the symbol "=" is used to assign a value to a variable, the symbol "==" is used to test if two values are equals. Here we can see that 6 equal 3 is false.

2 - Truth tables

Operation between booleans are not the same as, for example, operation for numbers.

This is the results of the most common boolean expression that you need to know (no, and, or) when they are applied on two booleans and for each of the possible combinaison of value.

A no(A)
True False
False True
A B A and B
True True True
True False False
False True False
False False False
A B A or B
True True True
True False True
False True True
False False False

In JavaScript, these operators are in order: "!A" for no(A), "A && B" for A and B and "A || B" for A or B. So if we look at this code :

var A;
var B;
var result;
A = true;
B = false;
result = A && B;
console.log(result);

We get a result. Try to predict the result with the table above and change the values of A and B!
