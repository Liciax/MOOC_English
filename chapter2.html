Part 2 - First program "Hello World"

Let's start with the basics : click on the "Write code" zone just above this text; then just copy the code in the frame below and and paste it into the "JavaScript" tab.

alert("Hello World!");

Did you see that? The program just greeted you! But what happened exactly?

1 - What is this program doing ?

First let's write the algorithm corresponding to that program :

Algorithm My_First_Algorithm

Beginning
  Write("Hello World!");
End

Do you see now what you did? You told the computer "I want you to write "Hello World!" on the screen". The sentence "alert(X)" (with X a sentence between quotes) is just the way in JavaScript to explain that to the computer. Feel free to try to write something else!

2 - How it's work ?

The instructions
In algorithmic, a program is a serie of instructions ended with the symbole ';'. 

instruction1;
instruction2;
instruction3;
All this instructions are delimited by ";", and it's very recommend to write one instrution by line. In the same way it is important to indent the code to make it readable. For exemple : 
instruction_1;
  instruction_1_1;
  instruction_1_2;
instruction_2;
