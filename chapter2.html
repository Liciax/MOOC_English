1 - What is this program doing ?

For your first code, we print a message: Hello World. Why Hello World you said? It's a norm in all langage to first with this. So in JavaScript, you need to write that :
alert('Hello, World!');
alert is the key-word for show a pop up in your web browser.

2 - How it's work ?

The instructions
In algorithmic, for write your program you need write some instructions. 

instruction1;
instruction2;
instruction3;
All this instructions are delimited by ";", and it's very recommend to write one instrution by line. In the same way it is important to indent the code to make it readable. For exemple : 
instruction_1;
  instruction_1_1;
  instruction_1_2;
instruction_2;
