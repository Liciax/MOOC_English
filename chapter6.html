Part 6 - Loop structure

1 - While loop

It is sometimes necessary to perform a part of the algorithm several times, then used a repetitive structure.

This is the syntax of the while loop :

while (boolean expression) do
  instructions;
end while

You need to be aware of this points below when you use loops :

  • Check that the algorithm finishes (loop exit).
  • It is necessary that the Boolean expression can be calculated before the structure "while".
  • It is necessary that the Boolean expression is modified by the instruction list and may become wrong for avoid the infinite loop.

2 - For loop

This loop is an other loop which use a counter. A counter is incremented by 1 at every turn and takes all values between a start value and an ending value. It must not be changed in the loop.

This is the syntax of the for loop :

for counter from start value to ending value do
  instructions;
end for

Note :

It is also possible to add conditional structure inside loops and the same is true in the other way.

How to choose a loop ?

  • We know in advance the number of iterations → for loop
  • Otherwise if you are not sure → while loop
