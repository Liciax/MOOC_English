Part 6 - Loop structure

It is sometimes necessary to perform a part of the algorithm several times in a row. In that case you need a loop:

1 - While loop

This is the syntax of the while loop :

while (boolean expression) do
  instructions;
end while

and in JavaScript :

while (boolean expression) {
  instructions;
}

this while loop will check if the boolean is false and if it is not, it will execute the piece of code in it. After that, it will try again to check if the boolean is false and so forth. This will allow us to execute A piece of code as many time as we want. let's look at an example:

var A = 0;
var B = 2;
while (A <= B) {
  A = A + 1; //A grows bigger and bigger until it is as large as B
  console.log("A grows");
}
console.log(A);

At first, A = 0 < 3 so the piece of code is executed; the while evaluate the boolean again: A = 1 < 3 so the piece of code is executed again; the while evaluate the boolean again: A = 2 = 2 so the piece of code is executed once more; finally A = 3 > 2 so we get out of the while loop with A = 3.

You need to be aware of this points below when you use loops :

  • Check that the algorithm finishes (the piece of code in the while modify the values compared by the boolean) because otherwise the program will never finish : it will stay stuck in the loop forever.
  • It is necessary that the Boolean expression can be calculated before the structure "while" of course.

2 - For loop

The for loop is another loop which use a counter. this counter is a variable containing a number that is incremented by 1 at every turn and takes all values between a start value and an ending value. It must not be changed in the loop because it might create an infinite loop.

This is the syntax of the for loop :

for counter from start value to ending value do
  instructions;
end for

and in JavaScript :

for (i = start value; i <= ending value; i++) { 
  instructions;
}

let's look at an example, Can you guess what is does? :

for (i = 0; i <= 5; i++) { 
  console.log(i);
}

If you guessed that it would write every number between 0 and 5, well done!

Note :

It is also possible to add conditional structure inside loops and vice-versa, like so:

for (i = 0; i <= 5; i++) { 
  if( i > 3 ) {
    console.log("i is now bigger than 3");
  }
  console.log(i);
}

How to choose which loop you need ?

  • You know in advance the number of times you want to execute the piece of code → for loop
  • Otherwise if you are not sure → while loop
