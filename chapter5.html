Part 5 - Conditional structure

With Boolean we can determine if something is true or false. Now what if we wanted to substract a variable X to a variable Y, but only if Y was bigger than X? Let's look at conditionnals.

1 - Simple conditional

In algorithmic, the if conditional allows the program to execute piece of code (a serie of actions), but only if a condition is met

This is the syntax of the simple conditional if (notice the lack of ';' at the end of it) :

if (boolean expression) then
  instructions;
end if

and in JavaScript :

if (boolean expression) {
  instructions;
}

As we can see, if the boolean is true, the program execute the piece of algorithm that is between the "then" and the "end if" (or between the {} in JavaScript). If the boolean is false, we jump over this piece of code. Let's look at an example to try to understand how this works :

if (3 > 2) {
  console.log("3 is bigger than 2");
}

Here the condition in the if is met (3 is bigger than 2) and so we write the sentense in the console. On the other hand :

console.log("Is 3 smaller or equal to 2?");
if (3 <= 2) {
  console.log("Yes it is!");
}
console.log("Of course not!");

When the condition is not met, the piece of code in the if is ignored.

2 - Complex conditional

the simple if allows us to execute a piece of code only when a condition is met, but it is possible to execute another piece of code if the condition is not met. Only one of the two pieces of code will be executed depending on the condition.

This is the syntax of the complex conditional if else :

if (boolean expression) then
  instructions 1;                                  
else
  instructions 2;                             
end if

and in JavaScript :

if (boolean expression) {
  instructions 1;
}
else {
  instructions 2;
}

If the boolean is true, it will execute the instruction 1; on the other hand, if the boolean is false, it will execute the instruction 2. Let's look at a concrete example : 

if (5 <= 1) {
  console.log("5 is smaller or equal to 1");
}
else {
  console.log("5 is bigger than 1");
}

Let's try something a little bit more complicated: let's combine the code above with variables. Try to guess what it will write before executing it!

var A = 6 + 3;
var B = 2 + 8;
console.log("I like");
if (A > B) {
  console.log("coconuts");
}
else {
  console.log("oranges");
}
console.log("in the morning");

Finally, if you feel adventurous, try to modify the piece of code above so that it substract the smaller variable from the bigger variable! (remember that you can write variables in the console with console.log(X))
