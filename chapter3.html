Part 3 - Introduction to variable and type

Display something on the screen is great, but it's not doing to take us anywhere on it's own. The display is static and can't be changed:


alert("a");

alert("a");

alert("a");

alert("a");

this will print a 4 times in a row no matter what. Now let's have a look at variables.

1 - What are variables and how to use them

As the name imply, it is a data item that may take on more than one value during the runtime of the program. It is possible to display it multiple times during the program and not have the same result. Think of it as a container: you can put a data in it and at any time you can take it out and put another data in. Let's look at how to use it : first we need to state that we want to make a variable 


var namedvar;

the keyword "var" allows the computer to undersand that we want it to make a variable named "namedvar". There are 3 main types of data : - numbers (1, 2, 3.5) who are just numbers; - strings (a, hello, 3a2b) who are words composed of characters; - boolean who are variable that have only two values: True or false; In Javascript you not need to specify what kind of data you want to store Now we can start to use it :


var namedvar;
namevar = 1;
